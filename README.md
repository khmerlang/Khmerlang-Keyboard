## Features

- New keyboard layout for Khmer(show most type character on first page)
- Spell checker
- Roman to Khmer text
- Local word segmentation(using deep learning TFlite)
- Custom theme colors

Feature it doesn't have and probably will never have:
- Emojis
- GIFs
- Swipe typing
